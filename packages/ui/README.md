# @repo/ui

공통 디자인시스템 컴포넌트 패키지입니다.

## 주요 스크립트

```bash
pnpm --filter @repo/storybook dev
pnpm --filter @repo/storybook build
pnpm --filter @repo/ui lint
pnpm --filter @repo/ui typecheck
```

## Storybook

- 경로: `apps/storybook/.storybook`
- 스토리: `packages/ui/stories`
- 토큰/컴포넌트 스토리를 기준으로 앱 UI 정합성을 맞춥니다.
